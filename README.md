# MineVideos AI Skills Collection

MineVideos 项目的 AI Skills 集合，包含产品愿景、知识图谱等核心文档。

## 📦 Skills

### minevideos-vision
MineVideos 产品愿景文档，包含：
- AI 视频策划顾问平台定位
- 对话式探需流程设计
- Studio 页面三栏布局
- 技术架构和扩展性设计

### minevideos-knowledge-graph
MineVideos 项目知识图谱，包含：
- 项目架构（前端/后端/视频生成服务）
- 团队角色（@Eve/@Pixel/@Volt/@Forge）
- 核心功能和已知问题
- 失误率分析和改进计划

## 🎯 核心概念

**产品定位**：AI 视频策划顾问，不是视频生成工具

**核心价值**：在前期的文案策划阶段，通过对话式探需挖掘用户深层需求

**目标用户**：传统企业老板，表达能力有限但有深层需求

## 📊 流程图

```
产品主图 + 模糊需求
    ↓
Agent 互动探需（对话式挖需求）
    ↓
完整视频文案
    ↓
AI 生成分镜画板（Storyboard）
    ↓
用户确认/调整
    ↓
4K 高清画板 → Seedance → 成片
```

## 🚀 使用方式

这些 skills 可以被本地 agent 读取，帮助理解产品愿景和设计思路。

```bash
# 克隆仓库
git clone https://github.com/Link499/minevideos-skills.git

# 查看 skills
ls minevideos-skills/
```
