# MineVideos - AI 视频策划顾问平台

## 产品定位

**不是视频生成工具，是 AI 视频策划顾问**

核心价值在前期的文案策划阶段，不在视频生成阶段。

---

## 用户画像

**目标用户：传统企业老板**

特征：
- 不知道如何表达和输出自己的想法
- 只能上传产品主图 + 简单描述（如"帮我生成一个产品宣传视频，内容要有材料和人物带货"）
- 表达能力有限，但内心有更深层的需求
- 需要被引导才能说出真正想要什么

**典型场景：**
```
用户上传一张保温杯产品图
说："帮我生成一个产品宣传视频，内容要有材料和人物带货"

但他的真实需求可能是：
- 双层真空，12小时保温的功能展示
- 户外爱好者、上班族的目标受众
- 自然光/氛围光的光影偏好
- 具体的使用场景（登山、办公室、通勤）
- 品牌调性和视觉风格
```

---

## 核心理念

### 1. 对话式探需，不是选 ABC

传统工具：用户选风格 → 生成视频
MineVideos：AI 逐层引导 → 挖掘深层需求 → 生成完整方案

**探需维度：**
- 产品功能亮点
- 外观设计和材质
- 使用场景
- 目标受众
- 品牌调性
- 光影效果偏好
- 人物带货风格
- 材料展示重点
- 时长和节奏

### 2. 分镜画板先行，不是直接生成视频

流程：
```
产品主图 + 模糊需求
    ↓
Agent 互动探需（对话式挖需求）
    ↓
产品功能？外观亮点？使用场景？
    ↓
目标受众？品牌调性？光影偏好？
    ↓
人物带货风格？材料展示重点？
    ↓
完整视频文案
    ↓
AI 生成分镜画板（Storyboard）
    ↓
用户确认/调整
    ↓
4K 高清画板 → Seedance → 成片
```

### 3. AI 助理平台，视频只是第一个能力

架构天然支持扩展：
- 左栏加新 Agent（翻译助理、数据分析等）
- 中栏切换对话模式
- 右栏切换输出类型

```
│            │                          │                          │
│ 📊 数据分析 │                          │                          │
│ [规划中]   │ 用户: 户外爱好者、上班族 │                          │
│            │                          │                          │
│ ─────────  │ AI: 光影效果有偏好吗？   │                          │
│ 系统状态   │ 自然光/氛围光/studio?    │                          │
│            │                          │                          │
│ Provider ● │                          │                          │
│ 健康 3/3   │ [输入消息...] [📎] [→]   │ [确认文案] [生成分镜]    │
├────────────┴──────────────────────────┴──────────────────────────┤
│ 流程进度: 需求探询 ✓ → 文案生成 ✓ → 分镜预览 → 视频生成        │
└─────────────────────────────────────────────────────────────────┘
```

### 各栏职责

**左栏：Agent 面板**
- 从 Provider 选择 → Agent 能力面板
- 视频策划是当前激活的
- 其他 Agent 占位，未来可插拔
- 系统状态监控

**中栏：对话交互区**
- 从 prompt 输入 → 对话式需求挖掘
- AI 主动提问，逐层引导
- 不是选 ABC，是真正的对话
- 支持图片上传

**右栏：多阶段输出**
- 从视频预览 → 多阶段输出
- 文案 → 分镜 → 成片，逐步解锁
- 历史版本管理
- 确认和调整

**底部：量子纠缠进度条**
- 5 个节点：需求探询 → 文案生成 → 分镜预览 → 视频生成 → 完成交付
- 节点间有流动的光子粒子
- 已完成的段速度快亮度高，未激活段暗淡
- 当前步骤节点呼吸脉冲效果
- 每完成一步，量子态向右推进一次

## 对话流程设计

### 阶段 1：需求探询

```
用户上传产品图 + 简单描述
    ↓
AI 逐轮提问（5轮左右）：
- 产品卖点是什么？
- 目标受众是谁？
- 想展示哪些使用场景？
- 光影风格偏好？
- 时长和节奏要求？
    ↓
生成需求摘要
    ↓
用户确认
```

### 阶段 2：文案生成

```
基于需求摘要
    ↓
AI 生成完整视频文案
    ↓
用户确认/调整
    ↓
锁定文案
```

### 阶段 3：分镜预览

```
基于文案
    ↓
AI 生成分镜画板（每帧画面）
    ↓
用户预览每一帧
    ↓
确认/调整
    ↓
生成 4K 高清画板
```

### 阶段 4：视频生成

```
4K 画板 → Seedance
    ↓
生成视频
    ↓
用户预览
    ↓
导出/分享
```

## 技术架构

### 前端
- React/Vue
- 三栏布局
- 对话式交互
- 分镜画板预览
- 量子纠缠进度条（动画）

### 后端
- Node.js/Express
- 对话管理（多轮对话状态机）
- 文案生成（DeepSeek/Gemini/本地算力）
- 分镜生成（Image 模型）
- 视频生成（Seedance API）

### AI 能力
- **对话式探需**：LLM 引导用户说出深层需求
- **文案生成**：基于需求生成完整视频文案
- **分镜生成**：Image 模型生成每一帧画板
- **视频生成**：Seedance 从 4K 画板生成视频

### 扩展性
- Agent 插拔架构
- 每个 Agent 独立模块
- 统一对话接口
- 统一输出管理

## 关键转变

### 从工具到顾问

| 维度 | 传统视频工具 | MineVideos |
|------|-------------|-----------|
| 用户输入 | 选风格 + prompt | 对话式探需 |
| 核心价值 | 视频生成能力 | 前期文案策划 |
| 用户引导 | 无 | AI 逐层引导 |
| 输出阶段 | 直接视频 | 文案→分镜→视频 |
| 扩展性 | 单一功能 | Agent 平台 |

### 从生成到策划

**价值在前期的文案策划阶段，不在视频生成阶段**

用户真正需要的：
- 不是"帮我生成视频"
- 是"帮我想清楚我要什么视频"

AI 的角色：
- 不是执行者
- 是策划顾问
